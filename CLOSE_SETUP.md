## Features

The "Search IDs" section allows you to:
- View all Lead Custom Fields from your Close CRM account
- Search custom fields by name, ID, or type
- See field details including:
  - Field ID
  - Field name
  - Field type
  - Whether it accepts multiple values
  - Whether it's required
  - Available choices (for choice fields)
  - Editable by roles

## Usage

1. Navigate to "Search IDs" in the sidebar
2. Click on "Lead Custom Fields" tab
3. Use the search bar to filter custom fields
4. View detailed information for each field
